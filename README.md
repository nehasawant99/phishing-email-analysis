# Phishing Email Analysis

A collection of phishing email analysis cases documenting common phishing techniques, red flags, findings, and key security learnings.

# Objective

Practice identifying phishing emails by analysing real-world examples and documenting indicators of compromise.

# Cases Covered

- Look-alike Email Address
- Brand Impersonation
- OTP Verification Scam
- Fake Invoice Scam
- Fake Google Security Alert

# Analysis Process

1. Review the email.
2. Identify phishing indicators.
3. Document findings and red flags.
4. Determine the impact.
5. Provide recommendations.
6. Record key learning.

# Common Red Flags

- Fake sender address
- Look-alike domains
- Urgent language
- Unexpected attachments
- Suspicious links
- Requests for credentials or OTP
- Fake invoices or rewards
- Brand impersonation


# Skills Practised

- Phishing Detection
- Email Analysis
- Social Engineering Awareness
- URL Inspection
- Domain Verification
- Documentation and writeups

# Tools Used

- Google Phishing Quiz
- VirusTotal (https://www.virustotal.com)
- URLScan.io (https://urlscan.io)

# Learning

- Verify the sender's email address.
- Check links before clicking.
- Never share passwords or OTPs.
- Watch for urgency and impersonation tactics.
- Verify requests through official channels.
