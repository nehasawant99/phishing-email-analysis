## Phishing Email Analysis Project

## Overview
This project contains phishing email screenshots collected for cybersecurity awareness and phishing detection practice.

The purpose of this project is to identify common phishing indicators and improve awareness about social engineering attacks.



## Objectives
- Understand phishing techniques
- Identify suspicious email patterns
- Analyse fake login attempts
- Recognise social engineering tactics



## Common Indicators Found
- Fake sender addresses
- Urgent language
- Suspicious links
- Fake attachments
- Grammar mistakes
- Credential harvesting attempts



## Tools Used
- Screenshot Analysis
- VirusTotal
- Browser Link Inspection
- Manual Phishing Detection
- Google phishing Quiz



## Learning Outcome
This project helped improve understanding of:
- Phishing attacks
- Email security
- Cybersecurity awareness
- Social engineering techniques
